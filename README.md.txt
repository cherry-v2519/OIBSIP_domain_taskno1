Online Examination System in Java
Project Overview

This project is a console-based Online Examination System developed using Java. It demonstrates core Object-Oriented Programming (OOP) concepts and provides a simple examination platform with user authentication, password management, and automatic score calculation.

Features
User login authentication
Password update functionality
Multiple-choice question examination
Automatic score calculation
Result display after exam completion
Logout and re-login support
